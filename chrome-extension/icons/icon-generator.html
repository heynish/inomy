Inomy Icon Generator

Quick Instructions:

  1. Click each "Download" button below to save placeholder icons
  2. Or, use an image editor (Photoshop, Figma, Canva) to resize your design image to 16x16, 32x32, 48x48, 128x128
  3. Save them as icon16.png, icon32.png, icon48.png, icon128.png
  4. Place them in the chrome-extension/icons/ folder
  5. Reload the extension in chrome://extensions/
